# `PTxOut`

It represents a transaction output.

definition:

```ts
export const PTxOut = pstruct({
    PTxOut: {
        address: PAddress.type,
        value: PValue.type,
        datum: POutputDatum.type,
        refScrpt: PMaybe( PScriptHash.type ).type
    }
});
```

### `address`

type: `PAddress.type`


### `value`

type: `PValue.type`


### `datum`

type: `POutputDatum.type`


### `refScrpt`

type: `PMaybe( PScriptHash.type ).type`
