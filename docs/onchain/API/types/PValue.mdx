# `PValue`

It represents a ledger Value.

definition:

```ts
export const PAssetsEntryT = pair(
    PTokenName.type,
    int
);

export const PValueEntryT = pair(
    PCurrencySymbol.type,
    list( PAssetsEntryT )
);

export const PValue = palias(
    list( PValueEntryT )
);
```

The value implements all the properties defined on the [`TermList`](../../stdlib/TermList) utility term.
