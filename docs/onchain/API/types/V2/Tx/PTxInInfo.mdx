# `PTxInInfo`

:::info
The modifications in `PTxInInfo` from V1 version are highlighted.
:::


Type that represents detailed information about a Cardano transaction input. 
Each input references a UTXO and includes the resolved details of that UTXO, essential in understanding the funds being spent in the transaction.

Definition:

```ts
export const PTxInInfo = pstruct({
    PTxInInfo: {
        // highlight-start
        utxoRef: V1.PTxOutRef.type,
        // highlight-end
        resolved: PTxOut.type
    }
});
```
