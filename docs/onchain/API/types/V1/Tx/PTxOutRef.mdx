# `PTxOutRef`

Type that represents a reference to a UTXO.

Definition:

```ts
export const PTxOutRef = pstruct({
    PTxOutRef: {
        id: PTxId.type,
        index: int
    }
});
```

## `id`

Unique transaction ID

type: [`PTxId.type`](./PTxId)

## `index`

Index of the specific output within this transaction

type: [`TermInt`](../../../../stdlib/TermInt)
